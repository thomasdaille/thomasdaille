I’m Thomas, a data enthusiast exploring the foundations of **data engineering, analytics, and modern data workflows**.  

My GitHub is a place where I design and experiment with **pedagogical, fictitious projects** that help me test real-world techniques, tools, and architectures used across the industry.

## 🚀 What I Build

### 🧱 Data Engineering  
I create small-scale data pipelines and warehouse models to explore ingestion, transformation, and documentation practices.

- **Distrisys-Datawarehouse** — A fictitious end-to-end warehouse project where I model data, integrate sources, transform tables, and build Power BI dashboards with Microsoft Data suite.  

### 📊 Data Analysis 
I analyze datasets and build visual stories using BI tools.

- **[Data-Challenges](https://github.com/thomasdaille/Data-Challenges)** — My entry for the "DataDNA Dataset Challenge" - centered on car insurance policies and built in Power BI - designed to sharpen analytical thinking and problem-solving.
- **Sales-Product-Analysis** — A complete analytical workflow combining SQL, Python, and Power BI to understand product performance.

### 🤖 Machine Learning  
I explore ML workflows through practical, contained use cases.

- **HR-Analytics-Employee-Attrition** — A pedagogical project where I build a classification model to predict employee churn.

## 🎯 What These Projects are

- **Learning environments**, not production systems  
- **Fictional scenarios**, built to simulate real data workflows  
- **Hands-on practice**, where I validate concepts and tools  

I use these projects to **test**, **break**, and **understand** how modern data stacks behave — from ingestion to visualization. 

My goal is to build a strong foundation and progressively move toward more advanced, realistic pipelines.
